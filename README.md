# Raspberry_pi-used-blynk-with-http-request
how to interface raspberry pi with blynk 2.0, display an control devices
Here, we're using the HTTP request library without the Blynk library.

Open an account and create an interface on the Blynk platform. Retrieve the key to use in the code. On the Blynk application, create an interface with the virtual pins.
Run the code after replacing the key, then go to the application to control it.
